Stylized as "Ethos" (ethOS would've been boring... and was taken).

A companion app for ETH students, bundling functionality from various sources into one convenient place.

The app provides features such as course schedules, grade calculations, and more, all accessible through a user-friendly interface.

## Features (planned)
- View and manage course schedules
- Access course details and materials
- Calculate and track grades
- Grade prediction based on current performance
- Mensa menu access
